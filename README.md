# MERN CRUD Example

## Stack

- React.js
- Redux
  - Redux-thunk
- Node.js
  - Express
- MongoDB
  - Mongoose

## Feature

- Create Post
- See Detail Post
- Remove Post
- Edit Post
- Pagination
- Login & Register with JWT
- Backend: Node.js(Express) + MongoDB
- Design with Reactstrap(bootstrap v4)
